Detailed Description

The reverb.c file implements an artificial reverberation tail that is added to the output of the auralisation generated by the audio.c part of the auralisation process, to simulate the late reflections that the geometry.c part of the auralisation process could not determine in time.

The reverb algorithm currently implemented is the simplest binaural feedback delay network, with the customary recursive comb filter, and is illustrated in the image below. References: Udo Zolzer, "Digital Audio Signal Processing", 2nd Edition, Section 6.3.1 Schroeder Algorithm.
