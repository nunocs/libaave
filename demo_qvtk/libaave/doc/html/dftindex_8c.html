Detailed Description

The discrete Fourier transform (DFT) implementation in dft.h stores the Fourier coefficients in non-sequential order. This dftindex.c file implements a dft_index() function that returns the index into the calculated DFT data that corresponds to each Fourier coefficient.

Function Documentation

unsigned dft_index ( unsigned  i,
unsigned  n 
)

This function returns the index into the DFT data calculated by dft() that contains the Fourier coefficient i for a DFT of size n. n is a power of 2, up to the maximum supported by dft_index_table (currently 128). i is a value from 0 up to n / 2 - 1, since the input data is real:

  • X[0] = X[0].real; X[0].imag = 0;
  • X[1] = X[N/2].real; X[N/2].imag = 0;
  • X[N/2+i].real = X[N/2-i].real;
  • X[N/2+i].imag = - X[N/2-i].imag;
